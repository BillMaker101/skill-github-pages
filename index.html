Your Own Chess Code

a step-by-step project

from graphical user interface to state-of-the-art chess engine

designed in 2025 by author of 1980's chess program 'Sfinks'

This project holds the Microsoft Word documents which comprise ChessCode. ChessCode is a step-by-step process for creating a state-of-the-art chess program in Microsoft Visual Studio. ChessCode was designed to run on Intel 64-bit CPU's only. It uses Intel intrinsic instructions so may not run on other CPU's. It is written in 'C' with a little 'C++'.

The text in purple, "ChessCode Readme", is an active link to the ChessCode files. Run the mouse over it and you will see a link in the lower left of the window. Click on it to access the files. Once there, you will see Releases on the right. You can download the latest Release (.zip or .tar.gz) file to obtain your copy. Enjoy building your own chess game.
